PersonDetectionYolo — отслеживание проникновения на запрещённую территорию

Коротко: простая и понятная утилита на Python для детекции людей (YOLO) в видео и сигнализации при пересечении заранее размеченной запрещённой зоны. Код в одном файле main.py, зависимости в req.txt. 
GitHub

Что делает

Рисует на кадре запрещённую зону (интерактивно, мышью поверх кадра).

Сохраняет/загружает координаты зоны в restricted_zones.json.

Детектирует людей с помощью YOLO (ultralytics) и считает пересечения по точке «центр низа» бокса.

При попадании человека — на экране появляется красный alarm!. Тревога гаснет через 3 секунды после ухода.

Простая фильтрация ложных срабатываний (подтверждение по нескольким кадрам).

Быстрый старт

Клонируйте репозиторий и перейдите в папку:

git clone https://github.com/SalahidinAI/PersonDetectionYolo.git
cd PersonDetectionYolo


Установите зависимости:

python -m pip install -r req.txt


(в репозитории есть main.py и req.txt). 
GitHub

Положите в папку модель YOLO (например yolov8n.pt) и видео test.mp4.

Запустите:

python main.py

Управление (в программе)

В окне «Setup» — зажать левую кнопку и выделить прямоугольник зоны.

ENTER — сохранить зону и начать детекцию.

R — сбросить зону.

Q — выход.

P — пауза во время детекции.
